Nerd-Herd

This web app was created to help friends plan game nights. It includes unique rolling and polling mechanics to gamify making decisions. View the source code here!

Technologies

  • NodeJS
  • Express
  • MongoDB
  • JavaScript
  • Bootsrap

Demo

Homepage

The homepage allows the user to see herds (groups) they're a part of, view upcoming events, create herds, and create events

Events

In an event page, the user can say if they're going, create rolls and polls, and participate in rolls and polls

Rolling

Polling

Herds

In a herd page, the user can invite members, edit the herd, view events for the herd, and dispand the herd

Profile

From their profile page, a user can edit their personal information, view herds they're a part of, and delete their profile
